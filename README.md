## Vitegil-Server端

- 技术栈：SpringBoot + Mybatis Plus
- 基于 Jdk 1.8
- 成功部署到服务器
- 主要实现的功能：
  - 接收sdk监控到的信息：
    - 错误信息：js、promise、resource...
    - 设备信息：device
    - 性能信息：Performance
    - PV数量
    - ...
  - 将收集到的信息统计整理发送给前台：
    - 错误数量、类型、得分
    - 统计当天内每小时的错误数量、PV数量
    - 统计UV数量
    - 统计平均性能表现
    - ...
